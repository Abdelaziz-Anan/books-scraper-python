# 📚 Books Scraper (Python)

## 📌 Overview

This project is a simple web scraping notebook built using Python.
It extracts book data from the "Books to Scrape" website and saves it into a CSV file.

---

## 🎯 What This Project Does

* Scrapes book data from a single page
* Extracts:

  * Book title
  * Price
  * Rating
* Converts rating into numeric values
* Saves the data into a CSV file

---

## 🛠️ Tech Stack

* Python
* requests
* BeautifulSoup
* pandas

## ▶️ How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Open Jupyter Notebook:

```bash
python -m notebook
```

3. Run the notebook:

```
book_scraper.ipynb
```
